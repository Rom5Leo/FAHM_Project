# Field Asset Health Intelligence

End-to-end predictive maintenance / sensor ML portfolio project.

## Goal

Build a professional ML engineering project for sensor time-series data, anomaly detection,
degradation modeling, simulation, and engineer-facing tools.

## Project stages

1. Sensor/time-series pipeline
2. Physics/rule-based fault simulator
3. Maintenance copilot / RAG layer
4. Dashboard/API and optional multimodal extension
